Présentation

Après avoir fait plusieurs formulaires avec Bootstrap et en avoir essayé un avec Milligram, j'ai décidé de faire un comparatif d'un "formulaire type" avec les différents frameworks CSS existant sur le net.

Un framework CSS s'appelle en français un "framework d'interface" et en anglais un "front-end framework". Evidemment, "framework CSS" est beaucoup plus parlant.

Sur cette page, vous allez trouver le formulaire sans aucun CSS (ou alors juste un ultra minimaliste fait maison), ça vous permet donc de comparer avec tous les autres CSS trouvés et présentés. Tous les frameworks CSS vont avoir ce formulaire comme base et pas un autre. Certains vont demander des personnalisations que je vais apporter.

Sur chaque page, il y aura une fiche d'identité minimaliste pour chaque framework.

Si jamais vous découvrez qu'il manque un framework, n'hésitez pas à m'en faire part. Je désire être le plus exhaustif possible.

Le formulaire "type"

Ce formulaire est un modèle et bien sûr, il est non-fonctionnel. Il est juste là à titre d'exemple.
